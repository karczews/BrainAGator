Let's Learn & Play!

Brainagator is an educational brain-training app for children. Solve puzzles, match shapes, order numbers — all guided by a friendly voice.

View on GitHub Explore Games
Why Brainagator

Built for curious young minds

Every feature is designed to make learning feel like play.

🔊

Voice Guidance

Text-to-speech reads every instruction aloud, so even pre-readers can play independently.

🎮

Engaging Games

Seven brain-training mini-games that build shape recognition, number sense, and colour awareness.

🔒

100% Privacy

No accounts, no ads, no data collection. Your child's activity stays on the device, full stop.

🌍

Cross-Platform

One codebase powered by Kotlin Multiplatform runs natively on Android, iOS, Web, and Desktop.
